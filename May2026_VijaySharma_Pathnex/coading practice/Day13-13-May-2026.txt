# Matrix Builds

## Jenkins Pipeline - Matrix Build
You will learn how to **run the same build on multiple environemnts**

pipeline {
    agent any
    environment {
        INSTITUTE_NAME = "Pathnex"
    }
    stages{
        stage('Matrix Build'){
            matrix{
                axes{
                    axis{
                        name 'JAVA_VERSION'
                        values '8', '11', '17'
                    }
                }
                stages {
                    stage('Build') {
                        steps {
                            sh "echo Building with Java ${JAVA_VERSION}"
                        }
                    }
                }
            }
        }
    }
}

## GitLab CI - Matrix Jobs
You will learn how to **run jobs across multiple environments using matrix**.

stages:
  - build

build:
  stage: build
  image: maven:3.8.1-jdk-17
  parallel:
    matrix:
      - JAVA-VERSION: ['8','11','17']
  script:
    - echo "Building with Java $JAVA_VERSION"
