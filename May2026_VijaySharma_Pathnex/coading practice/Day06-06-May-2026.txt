# Debugging Pipelines
## Jenkins Pipeline - Debugging
You will learn how to **fix broken stages and handle errors**

pipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'https://github.com/Pathnex/sample-java-app.git'
            }
        }
        stage('Build'){
            steps{
                sh 'mvn clean compile || { echo "Compile failed"; exit 1; }'
            }
        }
        stage('Test'){
            steps{
                sh 'mvn test || {echo "Test failed"; exit1;}'
            }
        }

    }
}

## GitLab CI - Debugging
You will learn how to ** Handle Job Failures and exit codes**

stages:
  - build
  - test

build:
  stage: build
  image: maven:3.8.1-jdk-17
  script:
    - git clone https://github.com/Pathnex/sample-java-app.git
    - cd sample-java-app
    - mvn clean compile || {echo "Compile failed"; exit 1; }

test:
  stage: test
  image: maven:3.8.1-jdk-17
  script:
    - cd sample-java-app
    - mvn test || { echo "Tests failed"; exit 1; }
