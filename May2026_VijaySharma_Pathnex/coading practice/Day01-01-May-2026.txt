# Day 01 - Basics

## Ansible Task - Install Nginx on Pathnex Server

---
- name: Install Nginx on Pathnex Sercer     # Name we can change it as per our need
  hosts: all                                # All hosts form the inventory file
  become: yes                               # For Sudo User

  tasks:
   - name: Install Nginx
     yum:                                   # Module yum
       name: nginx
       state: present

## Kubernetes Task - Create Nginx Pod

apiVersion: v1
kind: Pod
metadata:
    name: pathnex-pod
spec:
    contianers: 
      - name: web
        image: nginx:latest
        ports:
          - containerPort: 80

# Git Integration
## Jenkins Pipeline - Checkout Git
We will learn how to **checkout a Git repository** and list files.

pipeline {
    agent any
    stages{
        stage('Checkout'){
            steps{
                git branch: 'main' , url: 'https://github.com/Pathnex/sample-java-app.git'
            }
        }
        stage('List Files'){
            steps{
                sh 'ls -la'
            }
        }
    }
}

## GitLab CI - Checkout Git
We will learn how to **checkout a Git repository in GitLab CI**

stages:
  - checkout

git-checkout:
  stage: checkout
  script:
    - git clone https://github.com/Pathnex/sample-java-app.git
    - cd sample-java-app
    - ls -la
